# Review-us frame: predefined icons

Place **SVG files** here to offer them as icon options when users select the "Review us" frame.

- Each `.svg` file in this folder appears as a selectable icon (name without extension is used as the label, e.g. `booking.svg` → "Booking"). The file `default.svg` is used as the "Default" option and is not listed separately.
- Recommended size: about **200×80** viewBox (or equivalent) so the icon fits the frame area.
- Users can also upload their own image (JPG/PNG) in the app; these predefined options are in addition to that.

To add a new icon: add a new `.svg` file in this folder. It will be listed automatically on the create QR page when the Review us frame is selected.
